Basic Scenario

  • Major work in Pure Sports Analytics done on Hawk-Eye data of Austrailian Open. However, the work doesn't involve Computer Vision.
  • Problem involving Computer Vision have sourced game sequences, tournament-based work is missing.
  • Most Computer Vision problem in this space involves tennis or badminton, very little work on Table Tennis.

Issues

  • Hawk Data is not public, would have been useful for 3d-ball tracking.
  • Broadcast videos are raw, unstructed and not annotated.
  • Minimal camera panning.

Possible Directions

  • Hotstar doesn't provide full-match replay but has live streaming. ESPNWatch not available in India.
  • HD Videos can be downloaded from tennistv (Paid subscription though, 10EUR/Month).
  • No grandslams though, only ATP matches.
